This is an open-source project made to recreate osu!mania by ppy as a website.
The website is designed to be used at school in which you will be able to play what ever song you want as long as the current version supports it.
It is still in development and will be for the foreseeable future.
The website to go to is as of this moment, https://matrix-vector-test.vercel.app.
If https://matrix-vector-test.vercel.app does not work and you still want to play it then try going onto the website, https://p5js.org.
If https://p5js.org does not work I unfortunately can not help you with the issue as it do not know of any other website in which you can use the files from this github page to play the game.
If you can use https://p5js.org and it is not blocked by your school's administrators or teachers, then follow the following steps:

Step 1: Download all files from this github page

Step 2: If you are currently at the homepage of the website and not the code editor then please press the "start coding" button in the bottom left corner of the website.

Step 3: Press the arrow button next to the name of the file "sketch.js"

Step 4: Press the "+" button next to "Sketch Files", choose "upload files", and click the box with the text "Drop files here or click to use the file browser"

Step 5: Select all the files you downloaded from the github page and click "Upload" in the bottom right corner of the file browser

Step 6: Delete the pre-existing files named "index.html", "sketch.js", and "style.css"

Step 7: Press the play button at the top left hand side of the screen

Step 8: If the game is currently squished you can hold on the line between the game preview and the file preview to change how wide the game preview is

Step 9: If everything goes right, enjoy! And try not to make the game too public to avoid your school admins banning it
